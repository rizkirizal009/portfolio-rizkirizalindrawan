Study Case

Application design creation for cake shop

Overview

This cake application that I designed aims to offer convenience in ordering cakes from home. This application was created to answer the anxiety that users experience when they want to order cakes online and without having to come to the store. And here I also show how this application works through the UI/UX design that I have been working on.

Duration

This project takes about 6 weeks

Problem Statement

A cake shop that is starting to have customers who are lazy to leave the house and want delivery.

User Persona

i am trying to define the user via user persona and user journey map below.

User Journey Map

User Research

After I created a user persona and a user journey map, I got several pain points that occurred, namely:

1. Customers can't come every day to the cake shop because it's quite far from home

2. Customers are sometimes lazy to go to the cake shop because the queue is quite long

3. Sometimes customers don't have time to buy cakes to the shop in person because it collides with working hours

Problem

The problems that I found that are often faced by customers according to the user persona and user journey map above are :

1. Customers can't come every day to the cake shop because it's quite far from home

2. Customers are sometimes lazy to go to the cake shop because the queue is quite long

3. Sometimes customers don't have time to buy cakes to the shop in person because it collides with working hours

Solution

The solution I gave from the problem above is to create an application which is useful for cake shop customers who want to order cakes from home without having to bother coming to the store directly

Testing

After I made a prototype of my design then I proceeded to the testing stage where I used the Unmoderated Usability Testing method using Maze.co then followed up using Google Forms to collect feedback from users. As for the tasks given to users when using my application :

1. Do sign up and login

2. Choose menu

3. Enter order address

4. Checkout and payment

5. Logout on the app

Payment Page & Tracking Order Page
